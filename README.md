# DLsite-Play-Image-Downloader

捕获 DLsite Play Viewer 中的 blob 图片并打包下载为 ZIP

Capture blob images from DLsite Play Viewer and package them into a ZIP download



## 安装

该插件提供了 Chrome 扩展与 Tampermonkey 脚本两种版本。

由于目前未上架 Chrome 扩展商店与 Edge 扩展商店，安装 Chrome 扩展需手动解压并加载，操作相对繁琐。因此，我们更推荐用户优先选用 Tampermonkey 脚本版本。



Tampermonkey 脚本版本：[点此安装（需先安装 Tampermonkey ）](https://github.com/gunfub/DLsite-Play-Image-Downloader/raw/refs/heads/main/release/DPID-1.4--Tampermonkey.user.js)

Chrome 扩展版本：[点此下载 ZIP 压缩文件](https://github.com/gunfub/DLsite-Play-Image-Downloader/raw/refs/heads/main/release/DPID-1.4.0--Chrome.crx.zip)



## 使用

请将页面导航至首页或您希望开始处理的任意位置，点击“开始捕获”按钮。随后逐页浏览至末页或您期望终止的任意位置，点击“结束捕获”按钮。本插件将自动将所有捕获到的图像文件打包为 ZIP 压缩文件，并通过浏览器下载功能提供保存。
